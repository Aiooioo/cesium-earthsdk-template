# Cesium WebGIS app template based on Earth SDK 2.0
---

基于Cesium的WebGIS一张图应用开发脚手架模板，采用 Earth SDK 2.0 构建，包含了常见的一张图功能实现。

## 功能点覆盖
 - 地图基本工具
   - 缩放
   - 全景
 - 地图交互工具
   - 书签
   - 测量（点、距离、面积、高度、方位角）
 - 地图分析工具
   - 视域分析
   - 通视分析
 - 环境特效
   - 天气模拟
 - 数据面板
   - 数据分类
